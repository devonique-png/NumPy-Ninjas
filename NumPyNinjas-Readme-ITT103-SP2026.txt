PROGRAM DESCRIPTIONS

This program is a Point of Sale (POS) system that allows users to input items, calculate totals and process transactions such as: viewing available items, adding items to cart, removing items from cart, viewing the users cart and checking out.

HOW TO RUN THE PROGRAM

Install a Python readable software on your computer. (Eg. PyCharm)
2. Download the .py file from GitHub.
3. Open the file.
4. Run the program.
5. Follow the program prompts.
